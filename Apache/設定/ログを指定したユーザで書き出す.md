# ログを指定したユーザで書き出す

```clike
CustomLog "|su -l taro -c \'rotatelogs /var/www/logs/access_log.%Y%m%d 86400 540\'" combined
ErrorLog  "|su -l taro -c \'rotatelogs /var/www/logs/error_log.%Y%m%d 86400 540\'"
```

suで指定するコマンド-c以降はエスケープした引用符（￥’）でくくる。
