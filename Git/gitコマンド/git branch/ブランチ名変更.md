# ブランチ名変更

###  現在のブランチのブランチ名を変更

```bash
git branch new_name
```

ブランチ名がnew_nameに変わった。

### あるブランチのブランチ名を変更

```bash
git branch -m old_name new_name
```

ブランチ名がold_nameからnew_nameに変わった。

### すでに存在するブランチ名で上書き

```bash
git branch -M new_name
```

## 注意点

- ブランチ名の変更はいつ行っても良い。reflogが書き換わるので。
