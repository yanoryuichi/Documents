# ファイルの修正を取り消す

### ファイルの修正を取り消す

```bash
vi test.txt           # (1) test.txtを修正する
git reset --hard HEAD # (2) (1)の修正を戻す
                                test.txt以外のファイルを修正していたり、ステージングしていても、元の状態に戻る
```

### 修正したファイルを元のファイルに戻す

```bash
vi test.txt               # (1) test.txtを修正する
git checkout test.txt     # (2) (1)の修正を戻す
```
