# ローカルブランチをリモートブランチで入れ替える

## ローカルのmasterをリモートのamsterで入れ替える

```bash
git fetch origin 
git reset --hard origin/master
```

### 事前にローカルのmasterをバックアップしておく

```bash
git commit -a -m "Saving my work, just in case"
git branch my-saved-work
```
