# アトリビュート

## 属性の設定・取得
### attr()

```clike
$('a').attr('href', 'http://www.yahoo.co.jp/'); // 設定
var url = $('a').attr('href');                  // 取得
```

### value属性の設定・取得

```clike
$('input').val('ABC');                          // 設定
var value = $('input').val();                   // 取得
```

## 要素にテキスト/HTMLを設定
### html()

```clike
$('span').html('<a href="http://www.yahoo.co.jp/">yahoo</a>');
```

### text()

```clike
$('span').text('yahoo');
```

## 周囲への挿入

```clike
$('span').wrap('<strong></strong>');
```
