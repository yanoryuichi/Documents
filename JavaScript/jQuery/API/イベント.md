# イベント

## クリックイベント
### 設定

```clike
$('a').click(function(){
   $(this).css('color', 'red');
});
```

### 実行

```clike
$('a').click();
```

## マウスオーバー
### mouseover() / mouseout()

```clike
$('li')
.mouseover(
    function(){
        $(this).css('color', 'red');
    })
.mouseout(
    function(){
        $(this).css('color', 'blue');
    }
 );
```

### hover()

```clike
$('li')
.hover(
    function(){
        $(this).css('color', 'red');
    },
    function(){
        $(this).css('color', 'blue');
    }
 );
```
