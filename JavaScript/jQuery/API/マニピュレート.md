# マニピュレート

## コンテンツの挿入
### 要素の前にコンテンツを挿入

```clike
$('h1').before($('<p>').text('BEFORE H1'));
```

### 要素の後にコンテンツを挿入

```clike
$('h1').after($('<p>').text('AFTER H1'));
```

### 子要素として末尾にコンテンツを追加

```clike
$('ul').append($('<li>').text('LAST ITEM));
```

### 子要素として先頭にコンテンツを追加

```clike
$('ul').prepend($('<li>').text('FIRST ITEM'));
```

## 要素の削除
### ある要素を削除

```clike
$('li').remove();
```

### ある要素の中を削除

```clike
$('li').empty();
```
