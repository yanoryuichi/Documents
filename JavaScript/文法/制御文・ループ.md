# 制御文・ループ

## 制御文
### if

```clike
if ( ... ) {
  ...
} else if ( ... ) {
  ...
} else {
  ...
}
```

```clike
if ( ... ) console.log( "abc" );
```

### switch

```clike
var num = 0;
switch ( num ) {
  case 0:
    console.log( "0" );
    break;
  case 1:
    console.log( "1" );
    break;
  default:
    console.log( "others" );
    break;
}
```
