# キャスト
## Number

```clike
var num = Number("12.3");   // 12.3
var int = parseInt("12.3"); // 12
```

## String

```clike
'' + 10 == '10'
```

## Boolean

```clike
!!'foo'; // => true
!!'';    // => false
!!'0';   // => true
!!'1';   // => true
!!'-1'   // => true
!!{};    // => true
!!true;  // => true
```
