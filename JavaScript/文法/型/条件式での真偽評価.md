# 条件式での真偽評価

## False

```clike
var a = null;
var b;                 // undefined
var c = 0;
var d = false;
var e = "";
var f = parseInt("A"); // NaN
```

以下の場合は偽で評価される。

- false
- 0
- ""
- NaN
- undefiled 
- null

## True

```clike
var g = [];         
var h = {};         
var i = new Array();
var j = function () {};
```
