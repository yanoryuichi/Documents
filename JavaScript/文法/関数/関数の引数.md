# 関数の引数

- 引数は省略追加可能。
- メソッドオーバーロードは不可能。
- 引数はコレクション扱い。

```clike
function concat(a, b, c) { return a + b + c; }

concat(1,2,3);   // => 6
concat(1,2);     // => NaN
concat(1,2,3,4); // => 6
```

```clike
function concat() {
    var sum = 0,
        len = arguments.length;
    for (var i = 0; i < len; i++) {
        sum += arguments[i];
    }
    return sum;
}

concat(1,2,3);   // => 6
concat(1,2);     // => 3
concat(1,2,3,4); // => 10
```
