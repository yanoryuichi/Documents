# ピボットテーブルで連続した日付を作る

## ピボットテーブルの作成

```clike
#!/usr/local/bin/perl

print "create table pivot ( id int );\n";
for (1 .. 10) {
    print "insert into pivot values ( $_ );\n";
}
```

```clike
mysql my_db < pivot.pl
```

## 連続した日付を作る
2009年1月以降の毎月（1日）の日付を作る。

```clike
SELECT DATE_ADD('2009-01-01', interval id month) FROM pivot
```
