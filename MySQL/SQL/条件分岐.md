# 条件分岐

## case

```clike
SELECT CASE 1 WHEN 1 THEN "one" WHEN 2 THEN "two" ELSE "more" END;
```

## if

```clike
SELECT IF(2>1, 'YES', 'NO');  --'YES'
UPDATE users SET class = IF(age >= 20, 'adult', 'child');
```
