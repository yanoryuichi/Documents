# プロンプトの変更

```clike
$ export MYSQL_PS1="\u@\h[\d]> "
$ mysql -h db_srv -u root -p foo_db
root@db_srv[foo_db]> SELECT ...
```

```clike
mysql> prompt \u@\h[\d]>
```

```clike
$ mysql --prompt="\u@\h[\d]" -h db_srv -u root -p foo_db
```
