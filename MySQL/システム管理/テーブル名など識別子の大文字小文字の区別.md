# テーブル名など識別子の大文字小文字の区別

## 設定確認

```clike
mysql> show variables like 'lower%';
+------------------------+-------+
| Variable_name          | Value |
+------------------------+-------+
| lower_case_file_system | OFF   |
| lower_case_table_names | 1     |
+------------------------+-------+
```

- 大文字小文字を区別する：0
- 大文字小文字を区別しない：1

## 設定変更

/etc/my.cnf:

```clike
lower_case_table_names = 1
```
