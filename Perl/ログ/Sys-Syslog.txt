# Sys::Syslog

```clike
use Sys::Syslog;

$program = $0;
openlog "$program $$", 'ndelay', 'user';         # 'user'もしくはLOG_USER （ファシリティ）
syslog 'info', 'this is %s', ' a test message';  # 'info'もしくはLOG_INFO （レベル）
closelog;
```
