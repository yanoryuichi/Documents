# ロック

## EXCLUSIVE MODE
### 1. taroがロックを掛けて、INSERTを実行

```clike
taro=# SELECT * FROM t1;
 num
-----
   1
   2
(2 行)
```

```clike
taro=# BEGIN; LOCK TABLE t1 IN EXCLUSIVE MODE;
```

ロックを実行する。

```clike
taro=# INSERT INTO t1 ( num ) SELECT MAX(num) + 1 FROM t1;
```

INSERTを実行する。

```clike
taro=# SELECT * FROM t1;
 num
-----
   1
   2
   3
(3 行)
```

この時点でまだCOMMITしない。

## 2. hanaがINSERTを実行

```clike
hana=# SELECT * FROM t1;
 num
-----
   1
   2
(2 行)
```

SELECTは実行できるが、結果には1.のINSERTは反映されてない。

```clike
hana=# INSERT INTO t1 ( num ) SELECT MAX(num) + 1 FROM t1;
```

このINSERTはブロックされる。

### 3. taroがCOMMITを実行

```clike
taro=# COMMIT;
```

このCOMMITにより、2.のINSERTのブロックが解除される。

```clike
taro=# SELECT * FROM t1;
 num
-----
   1
   2
   3
   4
(4 行)
```

numの値"3"が1.の、"4"が2.のINSERTを反映している。
