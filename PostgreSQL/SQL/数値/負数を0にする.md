# 負数を0にする

## 目的
正数ならそのまま、負数なら0にする。

## 方法

### ABS()

```clike
SELECT (ABS(-10) + (-10)) / 2 AS result;

```clike
result
```

     - -----

```clike
     0
```

```

```clike
SELECT (ABS(10) + 10) / 2 AS result;

```clike
result
```

     - -----

```clike
    10
```

```

### CASE

```clike
UPDATE t1 SET num = (CASE WHEN num - 10 <= 0 THEN 0 ELSE num - 10 END);
```
