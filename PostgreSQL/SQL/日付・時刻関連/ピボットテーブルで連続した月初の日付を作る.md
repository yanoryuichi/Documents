# ピボットテーブルで連続した月1の日付を作る

## ピボットテーブル

```clike
SELECT * FROM pivot;
```

```clike
 id
-----
   0
   1
   2
   3
  ..
  ..
```

## SQL

```clike
SELECT ('2010-01-01'::date + (id || ' months')::interval)::date FROM pivot;
```

```clike
     date    
------------
 2010-01-01
 2010-02-01
 2010-03-01
 .....
 .....
```
