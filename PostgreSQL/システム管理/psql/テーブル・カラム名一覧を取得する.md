# テーブル・カラム名一覧を取得する

## テーブル名一覧

```clike
\dt
```

## （シェルから）テーブル名一覧取得

```clike
echo '\dtv' | psql -t -A -F ',' DB_NAME
```

## （シェルスクリプトで）テーブル名・カラム名取得

```clike
#!/bin/sh

DB_NAME='foo'
tables=`echo '\dtv' | psql -t -A -F ',' $DB_NAME | cut -f 2 -d ','`
for t in $tables; do
   cols=`echo "\\d $t" | psql -t -A -F ',' $DB_NAME | cut -f 1,2 -d ','`
   IFS='
'
   for c in $cols; do
       echo "$t,$c"
   done
done
```
