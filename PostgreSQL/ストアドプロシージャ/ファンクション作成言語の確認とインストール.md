# ファンクション作成言語の確認とインストール
## 確認

```clike
 SELECT * FROM pg_language;
```

## インストール
### シェルで

```clike
createlang -h 127.0.0.1 -d mydb -U taro plpgsql
```

### SQLで

```clike
CREATE LANGUAGE plpgsql;
```
