# 条件分岐

## CASE

```clike
SELECT CASE WHEN status = 1 THEN '表示' ELSE '非表示' END AS _status FROM user;
```

## COALESCE

```clike
SELECT COALESCE(email1, email2, 'NO EMAIL') FROM user;
```

NULLでない最初の引数を返す。全てNULLの場合はNULLを返す。
