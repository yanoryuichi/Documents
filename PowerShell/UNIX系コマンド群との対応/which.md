# which

## Get-Command コマンドレット

```powershell
Get-Command vi | ft Definition

 Definition                      
----------                      
C:\Users\taro\App\vim\vim.exe
```

## where コマンド（where.exe）

```powershell
where.exe vi
```

```powershell
C:\Users\taro\App\vim\vim.exe
```

where.exeはPowerShellコマンドレットではなく、コマンドプロンプトのコマンド。
