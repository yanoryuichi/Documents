# ヘルプの参照 - Get-Help

## コマンド名をワイルドカードで指定する

```powershell
PS C:\Users\taro> Get-Help Write-*

Name                              Category  Synopsis
----                              --------  --------
Write-Host                        Cmdlet    カスタマイズされた出力をホストに書き込みます。
Write-Progress                    Cmdlet    Windows PowerShell コマンド ウィンドウ内に進行状況を表すバーを表示します。
Write-Debug                       Cmdlet    デバッグ メッセージをコンソールに書き込みます。
Write-Verbose                     Cmdlet    テキストを詳細メッセージ ストリームに書き込みます。
Write-Warning                     Cmdlet    警告メッセージを書き込みます。
Write-Error                       Cmdlet    オブジェクトをエラー ストリームに書き込みます。
Write-Output                      Cmdlet    指定されたオブジェクトをパイプライン内の次のコマンドに渡します。コマンド...
Write-EventLog                    Cmdlet    イベントをイベント ログに書き込みます。
```
