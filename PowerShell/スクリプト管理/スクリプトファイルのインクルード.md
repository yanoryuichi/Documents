# スクリプトファイルのインクルード

- .（ドットコマンド）を使う。

```powershell
PS> notepad.exe scripts.ps1

-----------------------
function Echo-HelloWorld {
  echo "Hello World"
}
-----------------------

PS> . scripts.ps1
PS> Echo-HelloWorld
Hello World
```
