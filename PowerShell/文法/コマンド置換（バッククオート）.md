# コマンド置換（バッククオート）

Bashと同様に$()で出来る。

### Bashのコマンド置換

```powershell
Bash> echo "DEBUG:" $(date)
  => DEBUG: Thu Nov 27 11:10:17 JST 2014
```

### PowerShellのコマンド置換

```powershell
PS> Write-Host "DEBUG:" $(date)
PS> Write-Host "DEBUG: $(date)"
  => DEBUG: 11/27/2014 11:10:55
```
