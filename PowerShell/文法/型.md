# 型


## タイプキャスト
[String]や[Int][Double]等でタイプキャストする。

```powershell
PS C:\Users\taro> 2 * 3
6
PS C:\Users\taro> [String] 2 * 3
222
```

### ToString()

```powershell
PS C:\Users\taro> $i = 10
PS C:\Users\taro> $i.ToString();
10
```

## 型を調べる
### GetType()

```powershell
PS C:\Users\taro> $i.GetType()

IsPublic IsSerial Name                                     BaseType
-------- -------- ----                                     --------
True     True     Int32                                    System.ValueType
```
