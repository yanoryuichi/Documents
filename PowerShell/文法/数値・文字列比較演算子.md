# 数値・文字列比較演算子

## 文字列比較
### 基本

```powershell
"A" -eq "A"       # True
"A" -ne "Z"       # True
"Z" -gt "A"       # True
"A" -lt "Z"       # True
"Z" -ge "A"       # True
"A" -le "Z"       # True
```

### c オプション - 大文字小文字を区別する 

```powershell
"A" -ceq "a"      # False
"A" -eq  "a"      # True
```

cオプションを指定しないと大文字小文字は区別しない。

### i オプション - 大文字小文字を区別しない

```powershell
"A" -ieq  "a"     # True
```

### ワイルドカード（SQLのLIKE演算子相当）

```powershell
"ABC" -like "*A*"           # True
"ABC" -like "A??"           # True
"ABC" -like "A"             # False
"ABC" -notlike "*Z*"        # True   ※notオプション
```

c/iオプション指定可能。

### 正規表現

```powershell
"ABC" -match "^A"           # True
"ABC" -match "^[A-Z]{1,3}$" # True
"ABC" -notmatch "^Z"        # True   ※notオプション
```

c/iオプション指定可能。
