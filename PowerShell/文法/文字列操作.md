# 文字列操作

## 結合

```powershell
PS> "abc" + "def"
abcdef
```

## 置換

### -replace演算子

```powershell
PS> "abcdef" -replace "abc", "ABC"
ABCdef
```

### Replaceメソッド

```powershell
PS> "abcdef".Replace("abc", "ABC")
ABCdef
```

### 正規表現

```powershell
PS> "abc123" -replace "([a-z]+)\d{3}", '000$1'
000abc
```

- 上は量数指定{3}と前方参照$1の例

### 1回だけ置換

```powershell
PS> "123123123" -replace "1", "0"
023023023
PS> "123123123" -replace "1", "0$1"
023023023
```

### Trim()で文字列前後のホワイトスペース除去

```powershell
PS brazil> " abc ".Trim()
abc
```

## 配列の中の文字列を結合（join）

```powershell
PS> $strs = @("abc", "def")
PS> $strs -join ""
abcdef
```

## 文字列を分割して配列へ（split）

```powershell
PS> $nums = "2016-04-01" -split "-"
PS> $nums[0], $nums[1], $nums[2]
2016
04
01
```

## ヒアストリング

### @" ... "@ - 変数展開する

```powershell

PS> $val = "japan"
PS> $str = @"
>> hello,
>> $val
>> "@
PS> $str
hello,
japan
```

### @' ... '@ - 変数展開しない

```powershell
PS> $val = "japan"
PS> $str = @'
>> hello,
>> $val
>> '@
PS> $str
hello,
$val
```
