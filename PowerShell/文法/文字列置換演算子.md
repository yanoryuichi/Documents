# 文字列置換演算子


## Trim() 文字列の前後の文字を除去
### ホワイトスペースの除去

```powershell
PS> " foo ".Trim()
foo
```

### 任意の文字列の除去

```powershell
PS> "'foo'".Trim("'")
foo
```

```powershell
PS> "12foo12".Trim("12")
foo
```

## 置換対象の文字列に＼バックスラッシュがある場合

### バックスラッシュを重ねる

```powershell
gci | % { $_.FullName -replace 'C:\\Users\\admin', '%USERPROFILE%' }
```

### Escapeメソッドを使う

```powershell
gci | % { $_.FullName -replace [regex]::Escape('C:\Users\admin'), '%USERPROFILE%' }
```

### Replaceメソッドを使う

```powershell
gci | % { ($_.FullName).Replace('C:\Users\admin','%USERPROFILE%') }
```
