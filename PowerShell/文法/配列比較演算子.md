# 配列比較演算子

## -contains 配列が指定したキーを含んでいるか

```powershell
PS> $names = @("taro", "jiro", "hanako")
PS> $names -contains "jiro"
True
```

## -in 値が指定した配列に含まれるか

```powershell
PS> $nums = @(1,3)
PS> 1 -in $nums
True
PS> 2 -in $nums
False
```

```powershell
PS> $source = @(1,2,3,4,5)
PS> $source | ? { $_ -in @(1,3) }
1
3
```
