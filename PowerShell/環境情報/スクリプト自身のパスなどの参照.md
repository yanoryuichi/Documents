# スクリプト自身のパスなどの参照

## $PSScriptRoot

- $PSScriptRoot を使う。

### 例 Microsoft.PowerShell_profile.ps1

```powershell
$moduleDir = (join-path $PSScriptRoot modules)
$env:PSModulePath = $moduleDir + ";" + $env:PSModulePath
```


## PowerShell 2以前

- コマンド名は$MyInvocation.MyCommandで取得できる。
