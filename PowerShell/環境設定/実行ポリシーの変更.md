# 実行ポリシーの変更


## 実行ポリシーの変更

```powershell
Set-ExecutionPolicy RemoteSigned
```

※ PowerShellを管理者として実行する事。

## 実行ポリシーの確認

```powershell
Get-ExecutionPolicy

RemoteSigned
```
