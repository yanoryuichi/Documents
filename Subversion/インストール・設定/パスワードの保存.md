# パスワードの保存

.subversion以下の設定ファイルに以下の設定を記述して、svn checkoutする。すると、パスワードが保存される。

### $HOME/.subversion/servers:

```bash
[global]
store-passwords = yes
store-plaintext-passwords = yes
```
