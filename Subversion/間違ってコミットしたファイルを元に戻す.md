#間違ってコミットしたファイルを元に戻す

## 0. diffでチェック

```bash
svn diff -r prev XXX.php
```

## 1.a. diffをとって、patchを当てる

```bash
svn diff -r prev XXX.php | patch -R
```

## 1.b.逆にマージする

```bash
svn merge -r head:prev XXX.php
```

## 1.c 古いファイルを取り直す

```bash
svn cat -r NN XXX.php > XXX.php
```

## 2. コミットする

```bash
svn commit XXX.php -m ''
```
