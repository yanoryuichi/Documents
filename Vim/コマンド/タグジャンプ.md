# タグジャンプ

## 移動
| C-] | カーソル下の単語のタグへジャンプ |
| C-t | 前のタグへ戻る |
| gC-] | 複数のタグがある場合に一覧から選んでジャンプ |
| C-w } | プレビューウィンドウを開いてジャンプ |
| ：pc | プレビューウィンドウを閉じる |

## カーソル下の単語（タグ）をプレビューウィンドウで開く

- カーソルを単語に合わせる。
- CTRL-W }または:ptaでプレビューウィンドウを開く。
- 必要であればCTRL-W Tでタブで開く。
- 必要でなければCTRL-W zまたは:pcでプレビューウィンドウを閉じる。

### タグ候補が複数ある場合

- CTRL-W }の代わりにCTRL-W g }とする。
