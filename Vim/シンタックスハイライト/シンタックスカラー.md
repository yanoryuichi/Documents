# 構文ハイライト


## Vimの表示色を確認

```clike
:so $VIMRUNTIME/syntax/colortest.vim
```

## 編集中のバッファの構文の指定色を確認

```clike
:so $VIMRUNTIME/syntax/hitest.vim
```

## 編集中のバッファの構文を確認

```clike
:syn list
```

## 色指定

```clike
:hi StatusLine ctermfg=white
```

## 未指定

```clike
:hi StatusLine guibg=NONE
```
