# ビジュアルモードで矩形選択範囲内だけを置換する

以下のようなテキストがあって、

```clike
2000/3/30
2001/4/30
2002/5/30
```

年部分だけをビジュアルモードで矩形選択して、

```clike
:'<,'>s/\%V0\%V/9/g
```

のように"\%V"で対象をくるむと、以下のように年の0だけ9に置換される。

```clike
2999/3/30
2991/4/30
2992/5/31
```

## 選択範囲内だけを検索する

- CTRL-Vで必要な箇所を選択する。
- ESCでビジュアルモードを抜ける。
- /で検索を開始する。
- \%Vfoo のように最初に\%Vを入力して、次に検索したいワード（ここではfoo）を入力する。
