# あらゆる言語に対してキーワードの補完を有効にする

```clike
if exists("+omnifunc")
 augroup omnifunc
   autocmd!
   autocmd Filetype *
           \	if &omnifunc == "" |
           \		setlocal omnifunc=syntaxcomplete#Complete |
           \	endif
 augroup END
endif
```
