# 読み込んだvimrcのパスを確認

## versionコマンド

```clike
:version
```

```clike
 system vimrc file: "$VIM/vimrc"
   user vimrc file: "$HOME/.vimrc"
    user exrc file: "$HOME/.exrc"
system gvimrc file: "$VIM/gvimrc"
  user gvimrc file: "$HOME/.gvimrc"
```

## vimrcに加えてプラグインなどのvimスクリプトファイルを表示

```clike
:scriptnames
```
