# コンテナの管理

## ”間違って削除されないようコンテナを保護する”で作ったOUを削除

- ウィンドウメニューの「表示」→「拡張機能」にチェックを付ける。
- 当該コンテナを右クリックしてプロパティを開き、「オブジェクト」タブを選ぶ。
