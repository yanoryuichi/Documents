# 変換キー・無変換キーの利用


```autohotkey
!sc07B::MsgBox, "CTRL-無変換"
sc07B & z::MsgBox, "無変換-z"
```

```autohotkey
!a::Send, {sc07B}
```

変換・無変換キーのVKやSCはタスクトレイのAHKをダブルクリックで開いて、メニューの[View] -> [Key history and script info]で確認する。

### キー設定

- sc07B => "無変換
- sc079 => "変換"
