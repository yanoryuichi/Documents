# タスクバーサイムネイルプレビュー

## プレビュー画像のサイズを変更

- レジストリエディタで以下を開く。

```powershell
HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Taskband
```

- MinThumbSizePxという名前でDWORDを新規作成する。
- 値を400（10進数）などにする。
