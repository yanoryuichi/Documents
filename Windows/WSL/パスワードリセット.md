# パスワードリセット


### 1. ログインユーザをrootへ変更

- WSLを終了する。
- 管理者権限でコマンドプロンプトまたはPowerShellを起動する。
- 以下のコマンドを入力して、ログインユーザをrootに変更する。

```powershell
lxrun /setdefaultuser root
```


### 2. パスワードリセット

- WSLを起動、ログインする。
- rootでログインしているはずなので、以下のコマンドを入力して、パスワードリセットしたいユーザのパスワードを変更する。

```powershell
passwd taro
```

### 3. ログインユーザを戻す

```powershell

lxrun /setdefaultuser taro
```
