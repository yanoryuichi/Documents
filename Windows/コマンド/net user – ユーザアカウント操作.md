# net user - ユーザアカウント操作

## 全ユーザ一覧

```powershell
net user
```

## ユーザ情報の表示

```powershell
net user taro
```

## アカウントの追加

```powershell
net user /add hanako
```

## ユーザパスワードの変更

```powershell
net user taro new_passwd
```
