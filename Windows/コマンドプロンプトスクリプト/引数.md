# 引数

### すべての引数を参照する

```powershell
notepad.exe %*
```

### 指定した引数を参照する

```powershell
notepad.exe %1
notepad.exe %2
```

以降、%9まで参照可能

### 引数をずらす（シフトする）

```powershell
if exist "C:\tmp\%1" set dir1="%1"
shift
if exist "C:\tmp\%1" set dir2="%1"
echo %dir1%
echo %dir2%
```

shiftを実行することで、%2の内容が%1にずれる。
