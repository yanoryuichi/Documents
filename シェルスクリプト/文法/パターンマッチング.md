# パターンマッチング

## 二重アスタリスク(*)でサブディレクトリにマッチさせる
### 今回のファイル構成

```bash
.
|-- A
|   `-- a.txt
|-- B
|   `-- b.jpg
`-- x.txt
```

### サブディレクトリ下の.txtファイルにマッチ

```bash
$ ls -1 **/*.txt
A/a.txt
```

### カレントディレクトリとサブディレクトリ下の.txtファイルにマッチ

```bash
$ ls -1 {,**/}*.txt
A/a.txt
x.txt
```
