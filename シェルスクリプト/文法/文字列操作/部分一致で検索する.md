# 部分一致で検索する


## grepを使って

```bash
if $(echo $TERM | grep xterm > /dev/null) ; then
    echo OK
fi
```


## Bashの正規表現マッチオペレーター（=~）を使って

```bash
TERM=xterm
if [[ "$TERM" =~ ^x ]]; then
  echo OK
fi
  => OK
```

```bash
TERM=xterm
if [[ "$TERM" =~ ^x ]] && echo OK || echo NO
  => OK
```

```bash
TERM=xterm
OS=Linux
[[ "$TERM" =~ ^x && "$OS" = "Linux" ]] && echo OK
  => OK
```

- ifに続く条件式は複合コマンド[[ ]]を使う。testコマンド[]ではダメ。
- 正規表現は引用符で包まない。"^x"はダメ。^xとする。
