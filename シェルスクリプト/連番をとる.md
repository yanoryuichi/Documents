# 連番をとる

## seqコマンドを使って

```bash
IFS=$'\n'; for i in `seq 1 3`; do echo "DEBUG: $i"; done
```

```bash
DEBUG: 1
DEBUG: 2
DEBUG: 3
```

## BASHのブレースエクスパンションを使って

```bash
for i in {1..3}; do echo "DEBUG: $i"; done
```

```bash
DEBUG: 1
DEBUG: 2
DEBUG: 3
```

```bash
for i in {0..10..3}; do echo "DEBUG: $i"; done
```

```bash
DEBUG: 0
DEBUG: 3
DEBUG: 6
DEBUG: 9
```
